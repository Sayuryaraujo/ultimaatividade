# Sistema de Gerenciamento Escolar

## Descrição do Projeto

Sistema desenvolvido para gerenciamento de professores, turmas, alunos e atividades escolares, permitindo o cadastro, consulta, edição e remoção de informações através de uma interface web integrada a um banco de dados.

# Tecnologias Utilizadas

* Visual Studio Code 

### Back-end

* JavaScript
* node
* prisma

### Front-end

* HTML
* CSS3
* JavaScript

# Servidor de Aplicação

* Node.js

# Banco de Dados

* MySQL
* XAMPP

# Dependências Utilizadas

* Express.js
* Prisma ORM
* MySQL2
* Cors
* Dotenv
* Nodemon
